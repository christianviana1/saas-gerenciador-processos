# Screenshots do PWA — Sistema Jurídico

Este diretório contém os screenshots exibidos no prompt de instalação do PWA em navegadores compatíveis (Chrome Android, Edge).

## ⚠️ Substituição Obrigatória

Os arquivos listados abaixo precisam ser criados com capturas de tela reais da aplicação antes do deploy em produção.

## Arquivos Necessários

| Arquivo                   | Tamanho   | Finalidade                              |
|---------------------------|-----------|-----------------------------------------|
| `screenshot-desktop.png`  | 1280×800  | Preview no prompt de instalação desktop |
| `screenshot-mobile.png`   | 390×844   | Preview no prompt de instalação mobile  |

## Dicas

- Capture telas representativas da interface principal (lista de processos ou dashboard).
- Use Chrome DevTools com emulação de dispositivo para capturar o tamanho exato.
- Formato obrigatório: PNG.
